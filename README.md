## 🚀 Featured Projects

### 🤖 Meeting Assistant — AI Agent Orchestration Desktop App
> **Electron.js · React.js · FastAPI · LangGraph · LangChain · Deepgram · OpenAI API · MongoDB Atlas · Redis · WebSockets**

A cross-platform desktop application (Windows & macOS) built with Electron.js and React.js, featuring a fully orchestrated multi-agent AI backend for intelligent meeting management.

- 🧠 **Multi-agent system** built with LangGraph — specialised agents for transcription, summarisation, action item extraction, follow-up drafting, and sentiment analysis with defined handoff logic
- 💾 **Short & long-term memory** — in-context window management + MongoDB Atlas Vector Search for persistent cross-session recall of decisions and action items
- 🎙️ **Real-time transcription** via Deepgram WebSocket API with speaker diarisation, feeding live chunks directly into the agent pipeline
- 🔧 **Tool-calling layer** enabling agents to autonomously draft follow-up emails, assign tasks, and create calendar events
- 📊 **Rich dashboard** with live transcription view, agent activity feed, meeting timeline, speaker breakdown, and searchable meeting history
- 🔐 **Full auth system** with login, registration, and protected routing inside the Electron shell
- ⚡ FastAPI async backend with Redis for agent state caching and session management
